## Controls
- Click game to lock mouse
- WASD: move
- Mouse: look
- Space: jump
- Shift: sprint
- Ctrl: crouch
- C: toggle 1st/3rd person
- 1/2/3: weapon slots
- R: reload
- E: interact / loot
- I: inventory
- Esc: pause / unlock mouse

## Game flow
Title → New Game → Intro (skippable) → Vault 811 → Exit → Wasteland (streaming tiles, enemies, loot)
